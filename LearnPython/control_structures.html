Your First Program

Let's start off by creating a short program that displays "Hello world!". In Python, we use the print statement to output text: 


                >>> print('Hello world!')
                Hello world!

Try it below

Printing Text

The print statement can also be used to output multiple lines of text. For Example:


                >>> print('Hello world!')
                Hello world!
                >>> print('Hello world!')
                Hello world!
                >>> print('Spam and eggs...')
                Spam and eggs...
